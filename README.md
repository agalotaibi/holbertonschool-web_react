# holbertonschool-web_react

## Description
This repository contains various TypeScript projects and tasks as part of the Holberton School curriculum. The goal is to master TypeScript fundamentals, interfaces, classes, namespaces, and advanced types.

## Directory Structure
- **task_0**: Creating an interface for a student and rendering a table.
- **task_1**: Building Teacher and Directors interfaces, functions, and classes.
- **task_2**: Advanced types, type predicates, and string literal types.
- **task_3**: Ambient namespaces and declaration files (.d.ts).
- **task_4**: Namespaces and Declaration merging.
- **task_5**: Brand convention and Nominal typing.

## Setup
Each directory contains its own `package.json`. To run a specific task:
1. Navigate to the task directory: `cd task_x`
2. Install dependencies: `npm install`
3. Build the project: `npm run build`
4. Start development server: `npm run start-dev`
